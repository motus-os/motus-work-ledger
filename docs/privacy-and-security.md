# Privacy And Security

Motus records consequential work facts, not worker activity streams.

Motus Work Ledger does not require:

1. keystroke capture,
2. screen recording,
3. raw transcript capture,
4. employee monitoring,
5. centralized data collection,
6. blockchain,
7. compliance guarantees.

## Evidence References

Receipts should reference evidence by URI and digest when possible. They should
not embed raw sensitive content by default.

If evidence is redacted or withheld, the receipt must say so explicitly.

## Sensitive Metadata

Actor, model, tool, instruction, path, and evidence metadata can be sensitive.
Implementations should give operators control over what is emitted.
