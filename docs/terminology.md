# Terminology

## Motus

The overall ecosystem for workflow-neutral work ledger and receipt tooling.

## Motus Work Ledger

The portable model for recording consequential human, AI, CI, and tool-driven
work as durable facts and verifiable receipts.

## Work Journal

The append-only event stream for one or more bounded runs.

## Work Ledger

The organized durable work truth derived from journaled facts.

## Work Receipt

The portable proof object projected from ledger facts. A Work Receipt shows what
happened, what evidence existed, what outcome was claimed, and what was
accepted.

## `/motus`

The reference implementation CLI/library. It is not required for another system
to emit Motus-compatible Work Receipts.

## Store

The `/motus` local implementation substrate for runs, events, kind versions,
locks, and projections.
