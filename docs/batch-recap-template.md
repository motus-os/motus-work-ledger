# Batch Recap Template

Use this template for every bounded Motus Work Ledger batch.

```text
Batch:
Goal:
Repos touched:
Work item / claim:
Files changed:
Validation run:
Evidence:
Confidence before:
Confidence after:
Decision:
Open risks:
Next batch:
```

Confidence should be evidence-backed:

1. use `97%+` only for deterministic technical claims after tests pass,
2. use `90-95%` for documentation or messaging after review,
3. do not assign `97%+` to market or adoption claims before external evidence.
