# Non-Goals

Motus Work Ledger is not:

1. a workflow engine,
2. an agent runtime,
3. a model router,
4. a hosted control tower,
5. a compliance product,
6. employee surveillance,
7. a blockchain,
8. a raw transcript archive,
9. a global authority ledger.

The ledger records consequential work facts. It does not own the workflow that
produced them.
